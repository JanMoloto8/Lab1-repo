Latest Project

I developed a password encryption and decryptilon tool in Python, which aims to improve security by shifting
characters in the alphabet, based on a user-provided password and a shift key. The main function is encrypting
a message by moving characters forward and backward in the alphabet according to the specified value, which is
derived from the password.
